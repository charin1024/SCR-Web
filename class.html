프로그래밍언어론/ 김용태 교수님

이 과목은 프로그래밍 언어 C를 이용하여 프로그래밍의 기초를 학습한다.

컴퓨터의 기본 구조와 프로그래밍에 대해 이해하고 순서도와 프로그램을 작성해본다.

Visual Studio를 이용한 예제 및 실습을 통해 이론에만 치우치지 않는 학습을 한다.


수업 진행

→ C언어에 대한 이론 학습 후, 직접 실습을 통해 이를 확인하고 코드를 확장한다.



미디어커뮤니케이션 / 김용태 교수님

과거 미디어의 진화과정을 이해하고, 현 시대의 니즈를 분석하여 미래의 미디어 커뮤니케이션을 전망한다.

웹 기획과 관련된 내용을 이해하고 웹 기획 과정에서 사용되는 인터페이스 기획과 설계를 이해한다.

사이트 기획과 인터페이스 기획, 디자인 기획으로 나누어 전문적이고 구체적인 실무 내용을 이해한다.


수업 진행

→ 강독 수업으로 진행되며 웹 기획과 인터페이스 기획을 위한 사용자 정의, 정보 수집, 콘셉트 수립 등을 배운다.



ICT와 4차산업혁명 / 정성혜 교수님

ICT를 기반으로 하는 제4차 산업혁명의 개념을 이해하고 각 산업의 메가트렌드를 조망한다.

4차 산업혁명에 의해 촉발되는 정치, 경제, 사회, 문화 영역과 각 산업영역에서의 변화와 도전을 탐구한다.

제4차 산업혁명을 이끄는 기술과 방법론을 분야별로 학습하여 다가오는 4차 산업혁명 시대에 필요한 경쟁력을 갖춘다.


수업 진행

→ 4차 산업혁명 기술에 대해 학습하고 조사하여 발표하는 시간을 가진다.
